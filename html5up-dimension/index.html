1. Medical Device - Military Helmet Project

The MindShield project aims to address the high prevalence of traumatic brain injuries (TBIs) among US military members by developing an EEG monitoring system integrated into military helmets. This system is designed to monitor brain health, detect TBIs, and provide real-time data to soldiers and military leaders, enhancing the safety and well-being of military personnel.

Objectives:
  • To create a durable, minimally invasive EEG monitoring system integrated into military helmets.
  • To monitor EEG signals for early detection of TBIs and other brain-related issues.
  • To provide real-time data transmission to medical teams for immediate intervention.
Approach:
  • Real-Time EEG Signal Processing: Advanced algorithms analyze EEG data for signs of abnormal brain activity, allowing for immediate action.
  • Long-term EEG Monitoring: Continuous EEG data collection provides a comprehensive overview of brain health over time.
  • Real-Time Data Transmission: Integrated Bluetooth/WiFi connectivity ensures instant sharing of EEG data with medical teams, even in remote locations.
  • Real-Time TBI Detection: The system uses predefined neural markers to detect TBIs in real-time.
Prototype Design:

The prototype includes an array of eight electrodes integrated into a military helmet, interfaced with an Arduino microcontroller for robust data acquisition and processing. The design ensures consistent electrode contact with the scalp and minimal interference with the helmet's structural integrity. The prototype also features a custom SLA mount for the Arduino and a microcontroller for storage solutions, facilitating ongoing signal processing without impacting helmet usability.

Results:
  • The prototype demonstrated high accuracy in capturing EEG signals, with less than 5% variance in most metrics compared to control data.
  • The BrainGrapher script provided real-time graphical representation of brain activity, validating the efficacy of the signal processing algorithms.
  • Practical tests, including attention measurement, meditation, and motor function testing, confirmed the helmet's ability to record brain activity during various cognitive and physical states.
  • A classification model for TBI detection showed a performance accuracy of 90%, demonstrating the prototype's potential in clinical settings.
Conclusion:

The MindShield project successfully developed a functional prototype of an EEG monitoring system integrated into a military helmet. The system offers real-time monitoring and detection of TBIs, providing crucial health data to enhance the safety and well-being of military personnel. The combination of rigorous testing protocols and advanced signal processing capabilities ensures the prototype's functionality and potential for broader applications in medical diagnostics and cognitive research.

2. Brain Computer Interface Project

The Brain-Computer Interface (BCI) project aimed to enhance the control of an EEG-based BCI decoder using machine learning and deep learning techniques. This study was conducted by a group of students at Carnegie Mellon University, focusing on improving the accuracy and efficiency of BCI systems.

Objectives:
  • To use machine learning and deep learning techniques to improve the control of an EEG-based BCI decoder.
  • To analyze the effectiveness of different classification algorithms in decoding EEG signals.
  • To explore the potential of convolutional neural networks (CNNs) for feature extraction and classification in BCI applications.
Approach:

The project involved conducting EEG-BCI experiments where subjects controlled a cursor on a screen through motor imagery tasks. Their brain activity was recorded using EEG caps, and various machine learning algorithms were applied to decode the signals.

Methods:
  • Preprocessing: EEG data was preprocessed to remove artifacts such as eye blinks using Independent Component Analysis (ICA).
  • Feature Extraction and Filtering: The data was filtered using a bandpass filter, and relevant features were extracted for further analysis.
  • Machine Learning Algorithms: Logistic Regression, Support Vector Machine (SVM), and Random Forest classifiers were applied to the preprocessed data to evaluate their performance in decoding EEG signals.
  • Deep Learning Implementation: EEGNet, a compact convolutional neural network (CNN) architecture, was employed for advanced feature extraction and classification of EEG signals.
Results:
  • The logistic regression model achieved an initial test accuracy of 0.47, which improved with further data segmentation and analysis.
  • The SVM model showed significant improvement after hyperparameter tuning, achieving a test accuracy of 0.82 for the left-right task.
  • The Random Forest classifier's performance varied across different task and session segments, highlighting the impact of task-specific characteristics.
  • EEGNet demonstrated high classification accuracy, particularly for the up-down paradigm, with a Percent Valid Correct (PVC) of 97.6% for subject 1.
Conclusion:

The BCI project successfully implemented machine learning and deep learning techniques to improve the control of an EEG-based BCI decoder. The use of advanced algorithms and CNNs like EEGNet significantly enhanced the accuracy and reliability of decoding EEG signals, paving the way for more effective BCI systems.

3. Bioprinting - Cell Encapsulation Device for Type-1 Diabetes

This project focuses on developing a cell encapsulation device for treating Type-1 Diabetes Mellitus using advanced bioprinting techniques. The device encases living β cells, providing a timely and accurate amount of insulin without complications. The study explores the use of alginate and collagen bioinks to create structurally distinct, porous devices.

Objectives:
  • To develop a cell encapsulation device using alginate and collagen bioinks.
  • To enhance the structural integrity and functionality of the encapsulation device.
  • To improve insulin delivery and cell viability within the encapsulation device.
Approach:
  • Bioink Formulation: Collagen type I and alginate were selected for their biocompatibility and ability to promote cell adhesion and growth.
  • CAD Design: Encapsulation structures were designed using Autodesk Fusion 360 with specific dimensions for optimal nutrient and insulin diffusion.
  • 3D Printing: Fused Deposition Modeling (FDM) and Freeform Reversible Embedding of Suspended Hydrogels (FRESH) printing were used to create the encapsulation structures.
  • Cross-linking and Post-Processing: Cross-linking agents were applied to stabilize the printed structures, followed by washing and sterilization.
  • Characterization and Evaluation: The printed structures were evaluated for their mechanical properties, porosity, and structural integrity.
Results:
  • Alginate yielded better structural integrity compared to collagen, though both bioinks faced challenges in maintaining structure.
  • The flat surface device and tube structure showed variability in porosity and structural weaknesses.
  • Microscopic evaluation indicated that irregularly shaped pores and rough surface texture could impact insulin diffusion and cell attachment.
Conclusion:

This study highlighted the potential of using alginate and collagen bioinks for 3D bioprinting of islet encapsulation devices. Despite challenges in maintaining structural integrity, the findings suggest a need for further refinement of bioink formulations and scaffold designs to improve the efficacy of Type-1 Diabetes treatment. Future work will focus on optimizing bioink properties and conducting comprehensive in vivo studies to evaluate the long-term viability and functionality of these devices.

4. Tissue Engineering - Non Liposomal Magnetite Nanoparticle for Scaffold-Free Magnetic Skeletal Muscle Engineering

This project focuses on the development of scaffold-free magnetic skeletal muscle engineering using non-liposomal magnetite nanoparticles (MNPs). The goal is to enhance cell viability and aggregation without the need for traditional scaffolding materials. By leveraging the magnetic properties of MNPs, we aim to create a conducive environment for skeletal muscle tissue engineering, potentially offering a more effective and biocompatible solution for muscle regeneration.

Objectives:
  • To investigate the cytotoxicity and biocompatibility of different MNP loading conditions.
  • To analyze cell viability and aggregation in response to external magnetic fields.
  • To perform simulations of magnetic force/flux to predict the behavior of MNPs in a magnetic field.
  • To compare the efficacy of scaffold-free muscle engineering with traditional methods.
Approach:
  • Hypotheses: Non-liposomal MNPs (10 nm diameter) will display no significant cytotoxic effects on C2C12 cells. PEG-functionalized magnetite nanoparticles will promote better cell aggregation and viability compared to non-functionalized MNPs.
  • Testing Cell Viability for Different MNP Loading Conditions: Various MNP concentrations were tested on C2C12 cells. Live/dead assay was performed to assess cytotoxicity. Cell seeding was done on magnetic plates with external magnetic fields to observe pattern-free scaffold formation.
  • Pattern-Free Scaffold Formation Induced by External Magnetic Field: Cells were exposed to magnetic fields with varying intensities and directions. Aggregation patterns were observed and compared to literature standards.
  • Simulation (FEA and Magnetic Force/Flux Calculation): Finite element analysis (FEA) was conducted to simulate magnetic flux density and predict MNP behavior. Magnetic force calculations were performed to determine the effect on cell velocity and aggregation.
Results:
  • No statistically significant difference in cell viability was observed across different MNP loading conditions.
  • PEG-functionalized MNPs displayed improved cell aggregation compared to non-functionalized MNPs.
  • Simulation results aligned with experimental observations, validating the predictive models.
  • Pattern-free scaffolds exhibited high cell viability and low cytotoxicity, demonstrating the potential for scaffold-free tissue engineering.
Conclusion:

The live/dead assay indicated that non-liposomal magnetite nanoparticles are biocompatible and suitable for scaffold-free skeletal muscle engineering. PEG-functionalized MNPs offer enhanced cell aggregation, suggesting their potential for improving tissue engineering outcomes. The combination of experimental and simulation approaches provides a comprehensive understanding of MNP behavior in magnetic fields, paving the way for future advancements in scaffold-free tissue engineering.

5. Eye Gaze Communication System for Differently Abled Person

This project developed a low-cost eye gaze communication system for individuals with severe disabilities, enabling them to communicate using eye movements. The system used a camera module and various image processing algorithms to detect and track eye movements, which were then translated into communication commands.

Objectives:
  • To create a low-cost communication system for differently abled individuals using eye movements.
  • To develop a graphical user interface (GUI) that allows users to navigate and select communication options using their eyes.
  • To implement a reliable method for detecting and tracking eye movements using image processing techniques.
Approach:
  • Face Detection: Used Viola-Jones algorithm to detect the face and facial landmarks to locate the eyes.
  • Iris Detection: Applied Hough’s circle algorithm to track iris movements and detect gaze direction.
  • GUI Development: Designed a GUI with communication options that users could select by gazing at the screen.
  • Hardware Integration: Interfaced all components, including the camera module and GSM module, with a Raspberry Pi for system control.
Results:
  • Successfully tracked eye movements and translated them into communication commands using the GUI.
  • Provided a reliable and low-cost communication method for individuals with severe disabilities.
  • System was able to send messages to caregivers and family members using the GSM module.
Conclusion:

The eye gaze communication system provided an effective and accessible method for differently abled individuals to communicate their needs. The project demonstrated the potential of using low-cost technology and image processing techniques to enhance the quality of life for people with severe disabilities. 6. Prediction of Anemia Using Machine Learning

This project aimed to develop a machine learning model to predict anemia based on various blood parameters. The primary goal was to improve the accuracy and efficiency of anemia diagnosis using computational techniques.

Objectives:
  • To utilize machine learning algorithms to predict anemia with high accuracy.
  • To analyze the relationship between different blood parameters and anemia.
  • To develop a user-friendly graphical interface for easy access to prediction results.
Approach:
  • Data Collection: Collected a dataset of 1421 patient records with attributes including gender, hemoglobin, MCH, MCHC, and MCV.
  • Pre Data Processing: Cleaned the data by removing null and duplicate values and performed data transformation.
  • Exploratory Data Analysis: Used various plotting techniques to visualize relationships between attributes.
  • Modeling: Applied four machine learning algorithms (Logistic Regression, Decision Tree, Random Forest, K Nearest Classifier) to the dataset.
  • Evaluation: Evaluated models using cross-validation, F1 score, and confusion matrix to select the best performing model.
  • GUI Development: Developed a graphical user interface using Tkinter for easy input and display of prediction results.
Results:
  • Logistic Regression achieved the highest accuracy of 95.32% among the tested algorithms.
  • The model demonstrated high precision and reliability in predicting anemia.
  • The GUI provided an intuitive interface for users to input data and receive immediate predictions.
Conclusion:

This project successfully implemented a machine learning model to predict anemia with high accuracy. The use of Logistic Regression proved to be effective, and the developed GUI made the system accessible and user-friendly. This approach has the potential to assist healthcare professionals in diagnosing anemia more efficiently.
